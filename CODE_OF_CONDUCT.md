# Code of Conduct

## Our Standard

为了让协作保持高效、友好和可持续，我们希望所有参与者都能：

- 尊重不同背景和经验水平的贡献者
- 以建设性的方式提出意见和分歧
- 关注问题本身，而不是针对个人
- 在讨论中保持专业、清晰和克制

## Unacceptable Behavior

以下行为不被接受：

- 人身攻击、侮辱、骚扰或歧视性言论
- 故意制造对立、羞辱他人或恶意否定贡献
- 发布他人的隐私信息
- 反复偏离主题、破坏协作流程

## Enforcement

项目维护者有权删除不当内容，并视情况限制或终止相关参与。

如果你遇到不当行为，请联系仓库维护者处理。
